# Lumos UI

Interface and landing layer for the **Lumos intelligence and automation ecosystem**.

---

## Overview

Lumos UI is the public-facing interaction layer of the Lumos ecosystem.

It is focused on:

- adaptive interfaces
- contextual interaction
- orchestration visibility
- human-centered automation
- modular AI experiences

---

## Current State

The project is currently in active development.

Some components and backend systems remain private while the architecture and orchestration layers continue evolving.

---

## Stack

- Astro
- TypeScript
- Vercel

---

## Development

```bash
npm install
npm run dev
```

---

## Vision

Lumos is being designed as an adaptive intelligence and automation environment that helps structure workflows, context, memory, and interaction in a more human-centered way.
